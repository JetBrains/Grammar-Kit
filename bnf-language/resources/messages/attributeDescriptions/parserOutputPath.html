A path to generate the parser file(s) in, resolved relative to the BNF file's parent directory. When set, the parser class's package is no longer used to derive the output directory; the explicit value wins.

Examples:


  {
    parserClass="com.example.MyParser"
    //Given that BNF file is located at "path/to/project/grammar.bnf"
    //Resulting parser file would be placed in "path/to/project/build/gen/com/example/MyParser.java"
    parserOutputPath="build/gen"
  }

CLI override: the headless generator accepts --parserOutputPath <path>. When both this attribute and the CLI flag specify different paths, the CLI value wins and a warning is printed. Pass --strict-paths to make such conflicts fatal.
