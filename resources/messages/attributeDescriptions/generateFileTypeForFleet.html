A list of parameters for IFileType generation. NOTE: Fleet only.
Option Default Value Description
fileTypeClass none The generated IFileType class qualified name. Note: Affected by adjustPackagesForFleet
debugName FILE Debug name of the root in AST tree.
languageClass none The qualified name of the Language class used in the generated IFileType implementation. Note: Affected by adjustPackagesForFleet

Examples:

IFileType generator settings: 

  {
    generateFileTypeForFleet=[fileTypeClass="some.filetype.psi.MyFileType" debugName="MyFileName" languageClass="some.language.MyLanguage"]
  }
Result:
