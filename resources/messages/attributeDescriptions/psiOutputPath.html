A relative path to generate PSI-files in. Used for separating PSI logic from parsers. It is required if a Syntax-API parser is intended to be KMP-compatible. (Kotlin-based generation only)

Examples:


  {
    //Resulting PSI files would be placed in "path/to/project/someOtherModule/output/gen"
    psiOutputPath="someOtherModule/output"
  }
