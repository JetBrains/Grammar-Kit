A path to generate PSI-files in, relative to the content root of the BNF file. Used for separating PSI logic from parsers. It is required if a Syntax-API parser is intended to be KMP-compatible. (Kotlin-based generation only)

Examples:


  {
    //Given that BNF file is located at "path/to/project/somModule/src/main/grammars/grammar.bnf"
    //Resulting PSI files would be placed in "path/to/project/someOtherModule/output/gen"
    psiOutputPath="../someOtherModule/output"
  }
